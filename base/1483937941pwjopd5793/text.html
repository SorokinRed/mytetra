Надо в поле ka_file_number добавить r_object_id номенклатуры

Поиск id номенклатуры

select r_object_id, ka_number from kc_filenomenclature_node

where ka_number like '19-01-33'

and ka_year >=date('01.01.2017 06:00:00')

Добавление в шаблон

update ks_doc object

set ka_file_number = '<id номенклатурного номера>'

where r_object_id = '<id шаблона>'
