Шаблон редактируется как документ, его необходимо открыть по прямой ссылке

например: http://172.16.0.204/webtop/action/properties?objectId=08000001830793b4

ID шаблона можно найти в поле ka_template_id существующего документа

  • заходим под пользователем
  • создаем проект документа по нужному шаблону
  • смотрим через DQL ka_template_id свеженького проекта
  • возвращаемся в учетку и жмем "отмена" для несоздания документа
