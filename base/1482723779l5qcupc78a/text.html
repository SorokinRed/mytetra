Шаблоны по структуре как обычные документы, хранится на стадии - "Проект документа"

Отличие заключается в том что в докуенте (ka_template_id) содержится id шаблона, а в шаблоне id самого себя.



Создание:

  1. Создаем подвид, если его нет (DQL)
    1. Создаем резерв номеров для подвида
  2. Создаем шаблон (DQL)
  3. Привязываем шаблон к подвиду (DQL)
  4. Предоставляем права на шаблон (API)
