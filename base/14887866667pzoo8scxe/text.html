У документа есть параметр ka_agree_description_index, отвечающий за этап, на который поступит документ.

То есть , если сделать вот такое обновление, то после отправки документа на согласование документ поступит на первый этап согласования.

update ks_doc object

set ka_agree_description_index = 0

where r_object_id='080000018d6fb4b7'

Если же вы возвращали документ на исправление, то обнуление параметра происходит автоматом.
