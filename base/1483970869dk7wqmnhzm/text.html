DQL Запрос добавление номера дела

create kc_filenomenclature_node object

set ka_dept_code = '1'

set object_name = 'filenomenclaturenode'

set ka_number = '17-20-14'

set ka_description = 'Книга учета рабочего времени'

set ka_filial = (select r_object_id from kc_department where object_name = 'Министерство по управлению государственным имуществом Свердловской области')

set ka_year = date ('01.01.2017 12:00:00')

set ka_is_active =1


К заметке прикреплен файл-шаблон для генерации запросов.

0й очищаем присланую номенклатуру, должны остаться 2 колонки - номер и наименование. Пустых строк быть не должно

1й столбец протягиваем название филиала

2й вставляем номер

3й вставляем название

4й протягиваем и копируем из него результат

Необходимо убрать кавычки вокруг запроса и прописать правильную дату, после этого проверяем на единичном и отправляем остальные
